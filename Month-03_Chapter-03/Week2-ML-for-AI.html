Random variables
Concept Focus
  • WHAT: A variable whose value is determined by a random process.
  • WHAT (continued): Discrete: takes countable values (dice: 1,2,3,4,5,6) Continuous: takes any value in a range (height: 1.5-2.1m)
  • WHY: Features in your dataset are random variables.
  • WHY (continued): Model predictions are random variables.
  • WHERE: Statistical modeling, probabilistic ML
  • KEY: "A variable whose value comes from a random experiment"
  • HOW TO APPLY: Create variable -> read/update value -> print value when needed.
  • USE IN PRACTICE: You use variables in almost every line of real code.
  • COMMON MISTAKE: Use clear names like total_marks, not vague names like x1.
  • EXAMPLE: Simple example: marks = 70, then marks = marks + 5, then print marks.
