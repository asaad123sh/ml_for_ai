Subtopic 1
Feedforward network
Concept Focus
  • WHAT: Layers of neurons where information flows in ONE direction: Input layer → Hidden layer(s) → Output layer No cycles or feedback.
  • WHAT (continued): Each layer fully connected to next.
  • WHY: Most fundamental neural network architecture.
  • WHY (continued): Can approximate any continuous function (universal approximation theorem).
  • WHERE: Tabular data, any general-purpose deep learning
  • KEY: "Data flows forward through layers — no loops"
  • HOW TO APPLY: Use flow: read example -> run example -> change one value -> compare output.
  • USE IN PRACTICE: You use it in real coding tasks when that exact job is needed.
